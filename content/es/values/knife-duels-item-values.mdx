Para entender realmente los **valores de los objetos en Knife Duels**, hay que observar los datos de rendimiento. No todos los cuchillos son iguales. Por el contrario, el **Butterfly Knife** es valorado por sus animaciones fluidas, que dificultan que los enemigos calculen el tiempo de sus bloqueos de manera efectiva.

<Section icon={Shield} title="Controles y Mecánicas de Combate">

Incluso el objeto de mayor valor es inútil si no sabes cómo empuñarlo. El sistema de combate en Knife Duels se basa en la física, lo que significa que tu movimiento y sincronización dictan si un golpe conecta. Dominar los controles es el "valor oculto" que mejora cada objeto de tu inventario.

| Plataforma | Entrada | Acción | Uso Estratégico |
| :--- | :--- | :--- | :--- |
| **PC** | **LMB** | Ataque Primario | Fuente principal de daño; requiere sincronización. |
| **PC** | **Shift** | Esprintar | Usado para cerrar distancias o provocar un ataque fallido. |
| **PC** | **Espacio** | Saltar | Evita barridos bajos y añade verticalidad. |
| **PC** | **Ctrl** | Bloqueo / Postura | Reduce el daño y permite realizar parries. |
| **Móvil** | **Joystick** | Movimiento | Esencial para "strafing" alrededor de los oponentes. |
| **Móvil** | **Icono de Toque** | Ataque | Los toques rápidos son mejores para realizar combos. |

<Callout type="tip" title="Táctica Avanzada: El Cebo">
Esprinta hacia tu oponente (Shift), luego detente de repente y retrocede justo cuando él ataque. Si falla, sus "frames de recuperación" lo dejan totalmente expuesto para un contraataque garantizado con un arma de gran alcance como la Katana.
</Callout>

La interacción entre los diferentes valores de las armas se hace evidente aquí. Un usuario de Daga debe usar el Esprint y el Salto para entrar en la "zona muerta" de un usuario de Katana. Si el usuario de Daga lo logra, su mayor velocidad de ataque (LMB) ganará el intercambio cada vez.

</Section>
