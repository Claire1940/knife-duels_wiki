<Section icon={Shield} title="Práticas de Troca Segura e Prevenção de Golpes">

À medida que os **valores de troca do Knife Duels** continuam a evoluir, o mesmo acontece com as táticas usadas por jogadores desonestos. Proteger seu inventário é tão importante quanto aumentá-lo. O sistema de trocas no Knife Duels foi projetado para ser seguro, mas o erro humano é a causa mais comum de perda de itens.

Antes de finalizar qualquer troca, você deve verificar o \"Status\" do item. Algumas skins podem parecer idênticas, mas pertencem a diferentes níveis de raridade. Sempre leve o seu tempo; um jogador apressando você para \"Aceitar\" é um grande sinal de alerta.

<Steps>
  <Step num="1" title="Iniciar a Troca">
    Encontre um jogador no lobby ou no discord da comunidade e envie uma solicitação de troca. Certifique-se de que ambos estão de acordo com os itens que serão trocados antes de abrir o menu.
  </Step>
  <Step num="2" title="Verificar Detalhes do Item">
    Passe o mouse sobre os itens oferecidos pelo outro jogador. Verifique o nome, raridade e quaisquer etiquetas específicas de \"Limitado\" para garantir que correspondam ao valor acordado.
  </Step>
  <Step num="3" title="Confirmação em Duas Etapas">
    O Knife Duels utiliza um processo de confirmação em duas etapas. Assim que ambos os jogadores clicam em 'Aceitar', uma janela de revisão final aparece. Use esse tempo para fazer uma última verificação.
  </Step>
  <Step num="4" title="Finalizar e Gravar">
    Após a conclusão da troca, é uma boa prática tirar um print do histórico de trocas. Isso ajuda em caso de falhas técnicas ou disputas.
  </Step>
</Steps>

| Sinal de Alerta | Descrição | Nível de Risco |
| :--- | :--- | :--- |
| **Pressa** | O jogador pressiona você para clicar em aceitar rapidamente. | Alto |
| **Troca de Itens** | O jogador remove um item de alto valor no último segundo. | Extremo |
| **Fora da Plataforma** | Pedidos para trocar por Robux ou itens externos. | Crítico |
| **Ghosting** | O jogador se recusa a mostrar o item no jogo primeiro. | Médio |

<Callout type="warning" title="Aviso Importante">
Nunca troque seus itens por \"promessas\" de recompensas futuras ou códigos. Uma vez que uma troca é confirmada no sistema, ela é irreversível. Atenha-se às trocas de item por item dentro do jogo para se manter seguro.
</Callout>

</Section>
